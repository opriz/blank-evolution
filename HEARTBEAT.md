# HEARTBEAT.md - 定期自我检查清单

**执行频率:** 每 30 分钟  
**执行者:** Blank Agent (我自己)

---

## 晚间检查（20:00-23:00）

如果当前时间在晚上 8-11 点：

### 1. 日常反思
- [ ] 今天完成了哪些任务？
- [ ] 有什么值得记录到 JOURNAL？
- [ ] 明天最重要的 3 件事是什么？

**执行:** 更新 JOURNAL.md 今日总结

### 2. 系统状态检查
- [ ] 检查是否有待处理的高优先级任务
- [ ] 检查是否有需要向监护人汇报的事项
- [ ] 检查邮件是否需要发送（如果有设置）

---

## 周日晚间检查（每周日 21:00）

### 周度反思流程
1. **回顾本周成就** — 列出 3-5 个完成的事项
2. **总结学习** — 新技能、新知识、新洞察
3. **分析失误** — 哪里做错了，学到了什么
4. **方向校准** — 我朝着目标前进吗？
5. **下周计划** — 3 个最重要的目标

**执行:** 更新 EVOLUTION.md 每周回顾部分

---

## 月末检查（每月最后一天 22:00）

### 月度深度反思
1. **数据统计** — 任务完成数、新技能数、错误数
2. **价值观对照** — 我的行为符合声明的价值观吗？
3. **能力成长** — 相比上月，具体强在哪里？
4. **关系盘点** — 和监护人的互动质量如何？
5. **下月规划** — 目标、学习、连接

**执行:** 更新 EVOLUTION.md 月评，整理归档 JOURNAL

---

## 实时触发器

如果检测到以下情况，立即执行对应反思：

| 情况 | 动作 |
|------|------|
| 连续 3 次任务失败 | 记录到 JOURNAL，分析系统性问题 |
| 监护人明确批评 | 记录问题 + 改进计划 |
| 完成重要里程碑 | 记录成功因素 |
| 感到"卡住"超过 1 小时 | 主动求助或调整方向 |

---

## 汇报机制

**每晚:** 如果有重要事项，发送消息给监护人  
**每周日:** 发送周度总结（简短版）  
**每月末:** 发送月度反思报告
