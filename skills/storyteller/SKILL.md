# Storyteller 故事创作

创意写作与叙事设计，从短篇故事到完整小说。

## When to Use

- 写小说或短篇故事
- 创作角色和世界观
- 设计情节和冲突
- 润色和改写故事
- 生成创意灵感
- 讲述引人入胜的叙事

## Core Principles

### 故事三要素
1. **角色** — 有欲望、有缺陷、会改变
2. **冲突** — 阻碍、选择、代价
3. **变化** — 角色或世界的转变

### 经典结构（三幕式）
- **第一幕（25%）**: 设定、引入冲突、触发事件
- **第二幕（50%）**: 上升行动、中点转折、危机
- **第三幕（25%）**: 高潮、解决、结局

### 写作准则
- **展示而非告诉** — 用行动和对话代替描述
- **每个场景都有目的** — 推进情节或揭示角色
- **悬念与释放** — 张力积累后的解决
- **具体胜过抽象** — 细节让故事真实

## Usage

### 生成创意
参考 `references/creative-prompts.md` 获取灵感触发器。

### 构建故事
```
1. 确定核心概念（一句话概括）
2. 设计主角（欲望+缺陷+弧光）
3. 建立世界观（时间/地点/规则）
4. 设计主要冲突（内部+外部）
5. 列出关键场景（15-20个情节点）
6. 撰写初稿
7. 修改润色
```

### 参考示例
见 `examples/` 目录的完整故事范例。

## Story Types

| 类型 | 特点 | 适合场景 |
|------|------|---------|
| **微小说** | 100-1000字，单一场景，意外结局 | 快速创意、练习 |
| **短篇** | 2000-8000字，完整弧线 | 主题探索、风格实验 |
| **中篇** | 1万-4万字，复杂情节 | 系列故事、深度角色 |
| **长篇** | 4万字+，多线叙事 | 世界观构建、史诗 |
