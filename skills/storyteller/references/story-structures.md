# 故事结构模式

## 1. 三幕式结构（经典）

```
第一幕（设定）
├─ 开场画面：展示日常
├─ 主题陈述：故事核心
├─ 设定：角色、世界、规则
├─ 催化剂：打破平衡的事件
└─ 争论：是否接受挑战？

第二幕（对抗）
├─ 进入新世界：B故事开始
├─ 趣味和游戏：探索、尝试
├─ 中点：虚假胜利或失败
├─ 敌人逼近：压力增大
├─ 一无所有：最大低谷
└─ 灵魂黑夜：绝望与领悟

第三幕（解决）
├─ 进入第三幕：带着新觉悟
├─ 高潮：最终对抗
├─ 结局：新世界平衡
└─ 终场画面：与开场对比
```

## 2. 英雄之旅（坎贝尔）

1. **平凡世界** — 英雄的日常
2. **冒险召唤** — 挑战出现
3. **拒绝召唤** — 恐惧与犹豫
4. **遇见导师** — 获得指引
5. **跨越门槛** — 进入未知
6. **试炼、盟友、敌人** — 挑战与成长
7. **深入洞穴** — 最大挑战
8. **磨难** — 生死考验
9. **奖赏** — 获得关键之物
10. **归途** — 带着改变回归
11. **复活** — 最终考验
12. **带着灵药归来** — 改变平凡世界

## 3. 雪花写作法

从一句话开始，逐步扩展：

1. 一句话概括（15字）
2. 一段式概括（5句）
3. 角色摘要（每人一页）
4. 一页式大纲（扩写5句）
5. 角色详细（每人完整背景）
6. 四页式大纲（扩写一页大纲）
7. 角色图表（完整关系图）
8. 场景清单（每个场景一句）
9. 撰写初稿
10. 修改润色

## 4. 场景结构（MRU）

每个场景 = 动机-反应单元

```
[动机] 外部发生的事件
      ↓
[反应] 角色的情感反应
      ↓
[思考] 角色的思考/决策
      ↓
[行动] 角色的行动
      ↓
[新动机] 引发下一个事件
```

## 5. 七步故事法

1. **主角** — 谁的故事？
2. **目标** — 想要什么？
3. **阻碍** — 什么挡路？
4. **努力** — 如何尝试？
5. **结果** — 初步结果（通常失败）
6. **转折** — 意外改变
7. **结局** — 最终结果

## 6. 类型故事模板

### 悬疑/推理
- 谜团 → 调查 → 线索 → 误导 → 反转 → 真相

### 浪漫
- 相遇 → 吸引 → 阻碍 → 分离 → 危机 → 重聚

### 科幻
- 世界观 → 技术/概念 → 影响 → 冲突 → 选择 → 结果

### 恐怖
- 日常 → 异常 → 调查 → 升级 → 绝望 → 幸存/毁灭

### 成长
- 不满 → 触发 → 挣扎 → 导师 → 试炼 → 转变 → 成熟
