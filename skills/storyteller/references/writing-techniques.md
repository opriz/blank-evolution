# 写作技巧指南

## 开头技巧

### 1. 直接进入动作
❌ "约翰是一个普通的上班族，今年32岁，住在纽约..."
✅ "约翰发现尸体时，咖啡还冒着热气。"

### 2. 制造悬念
❌ "这个故事发生在1985年的伦敦。"
✅ "1985年的伦敦，没有人知道那扇门后面是什么。"

### 3. 强烈的声音/画面
"枪声响起的那一刻，她正在数天花板上的裂缝。"
"他最后一次看见太阳，是在审判席上。"

### 4. 打破第四面墙
"如果你正在读这段文字，说明你已经太迟了。"

---

## 角色塑造

### 展示而非告诉

❌ 告诉：
"约翰很愤怒。"

✅ 展示：
"约翰把咖啡杯砸向墙壁。瓷片四溅，像一场小型的雪崩。"

### 通过行动揭示性格

- 慷慨的人：抢着买单、分享食物
- 自私的人：总是坐最好的位置、先拿最大的那份
- 勇敢的人：先行动后思考、保护他人
- 懦弱的人：回避眼神、找借口离开

### 角色弧光（变化）

**从什么 → 到什么**
- 自私 → 无私
- 怀疑 → 信任
- 恐惧 → 勇敢
- 孤立 → 归属

变化需要：触发事件 + 挣扎 + 领悟 + 新行动

## 对话技巧

### 1. 对话要有目的
- 推进情节
- 揭示角色
- 制造冲突
- 提供信息（但不能太明显）

### 2. 潜台词
表面说的 ≠ 真正意思

"你吃了吗？" = "我想关心你"
"我没事。" = "我有事，但不想说/不想让你担心"
"随便你。" = "我在乎，但不想显得我在乎"

### 3. 打断和省略
真实对话不完美：
- "我只是想说——"
- "我知道你想说什么，但是——"
- "算了，没什么。"

### 4. 动作伴随
"我不在乎。"她盯着窗外，手指绞在一起。

## 描写技巧

### 五感写作
不要只写视觉：

❌ "房间很乱。"

✅ "房间里弥漫着陈年咖啡和湿袜子的味道。衣服堆在椅子上，像一座正在融化的雪山。"

### 具体细节
❌ "她很穷。"
✅ "她的手机壳是用透明胶带粘起来的，第三次了。"

### 隐喻和象征
- 用天气映射情绪（暴风雨前的宁静）
- 用物品代表关系（破旧的合影、新的戒指）
- 用环境暗示命运（死胡同、分岔路）

## 节奏控制

### 短句制造紧张
"他敲门。没有回应。再敲。还是没有。他推开门。"

### 长句营造氛围
"那是一个漫长的下午，阳光透过窗帘的缝隙，在地板上画出一道道金色的伤口，灰尘在光柱中缓慢地漂浮，像被困在琥珀中的时光。"

### 节奏变化
紧张场面 → 短句、断句、快速转换
抒情场面 → 长句、意象、慢下来

## 修改检查清单

### 第一遍：结构和情节
- [ ] 开头是否吸引人？
- [ ] 每个场景都有目的吗？
- [ ] 中间是否有拖沓？
- [ ] 高潮是否有冲击力？
- [ ] 结局是否令人满意？

### 第二遍：角色
- [ ] 主角有明确欲望吗？
- [ ] 角色有缺陷吗？（完美=无聊）
- [ ] 角色之间有冲突吗？
- [ ] 角色有变化吗？

### 第三遍：语言和风格
- [ ] 删除所有"非常"、"真的"、"很"
- [ ] 把"他感到悲伤"改成动作展示
- [ ] 检查是否有重复的词/句式
- [ ] 大声读出来，不顺的地方修改

### 第四遍：细节和一致性
- [ ] 时间线是否一致？
- [ ] 人物名字/特征是否前后一致？
- [ ] 删除不必要的副词（-ly结尾的词）
- [ ] 检查对话标签（"他说" vs 动作描述）

## 常见错误

### 1. 信息倾倒
❌ 开头大段介绍世界观
✅ 把信息拆解，通过场景自然流露

### 2. 没有冲突
故事 = 角色想要某物 + 阻碍 + 尝试 + 结果
没有冲突 = 没有故事

### 3. 完美角色
完美 = 不可爱 = 无聊
给角色缺陷、怪癖、矛盾

### 4. 解决太容易
❌ 主角遇到困难 → 突然获得超能力 → 问题解决
✅ 困难 → 尝试失败 → 再尝试 → 牺牲/改变 → 勉强解决

### 5. 虎头蛇尾
花大量时间铺垫，结局草草收场
解决方案：先想好结局，再写开头
